# Inventory App

A Flutter mobile app for inventory management.

## Getting Started

1. Install dependencies:
```bash
flutter pub get
```

2. Update the API URL in `lib/utils/api_config.dart`

3. Run the app:
```bash
flutter run
```

## Folder Structure

- `lib/models/` - Data models
- `lib/providers/` - State management with Provider
- `lib/screens/` - UI screens
- `lib/services/` - API service layer
- `lib/widgets/` - Reusable UI components
- `lib/utils/` - Colors, configs, helpers
